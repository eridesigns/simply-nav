Simply Navigation

Lightweight simple navigation built with flexbox and vanilla JavaScript

Features

  • No dependencies.
  • Fully responsive.
  • Built using Flexbox & vanilla JavaScript only.
  • Menu button changes to a close button 'X' (CSS only @animated-burgers )
  • Mobile menu closes when a link is selected (Like the 'Last Section' link on this page).
  • Easy close mobile menu. Closable via 'X' and right side page overlay.
  • Mobile menu scrollable (if its content goes beyond the screen)
  • Page and body are not scrollable (when mobile menu open).

Customizable

  • Breakpoint variable - default at 800px
  • Sticky navigation bar available.

Sticky Nav is Off

Note: Sticky navigation bar requires padding and/or margin on the parent or adjacent element. For example, on this page the adjacent main HTML element has a top padding of 5.5rem (nav height is 4.5rem).
